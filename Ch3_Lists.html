Chapter 3: Lists


There are time when we need to use lists. HTML provides us with three types:

  • Ordered lists are lists where each item is numbered. A recipe, or legal contract are some examples.
  • Unordered lists are lists that begin with a bullet point (rather than characters that indicate order.)
  • Definition lists are made up of a set of terms along with the definitions for each of those terms.

Ordered Lists

  1. Chop potatoes into quaters
  2. Simmer in salted water for 15-20 minutes until tender
  3. Heat milk, butter, and nutmeg
  4. Drain potates and mash
  5. Mix in the milk mixture

Browsers indent lists by default. Sometimes there will be a type attribute used with the ol element to specify the type of numbering (numbers, letters, roman numerals, etc). It is better to use the CSS list-style-type property of this.

Unordered Lists

  • 1kg King Edward potatoes
  • 100ml milk
  • 50g salted butter
  • Freshly grated nutmeg
  • Salt and pepper to taste

Browsers indent lists by default. Sometimes there will be a type attribute used with th ul element to specify the type of numbering (circles, squares, diamonds, etc). It is better to use the CSS list-style-type property of this.

The description list usually consists of a series of terms and their definitions. Inside the dl element are pairs of dt and dd elements. The dt element contains the term being defined, the dd element contains the definition. Sometimes a list will have two terms for the same definition or two diffrent definitions for the same term.

Nested Lists

  • Mousses
  • Pastries
    • Croissant
    • Mille-feuille
    • Palmier

A second list can be put inside a li element to create a sub-list or nested list.

Browsers display nested lists indented further than its parent list. In nested unordered lists, the browser will usually change to the style of the bullet point too.

Lists

Scrambled Eggs

Eggs are one of my favorite foods. Here is a recipe for the deliciously rich scrambled eggs

Ingredients

  • 2 eggs
  • 1tbs butter
  • 2tbs cream

Method

  1. Melt butter in a fryin pan over a medium heat
  2. Gently mix the eggs and cream in a bowl
  3. Once butter has melted add cream and eggs
  4. Using a spatula fold the eggs from the edge of the pan to the center every 20 seconds (as is you are making an omlette)
  5. When the eggs are still moist remove from the heat (it will continue to cook on the plate until served)

An unordered list is used to outline the ingredients and an ordered list is used to describe the steps
